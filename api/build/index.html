u1m.jn.gl

This is an API into the awesome Umbrella Top 1 Million Domains dataset published by Cisco Umbrella.

Looking up the rank of a domain

$ curl http://u1m.jn.gl/domain/umbrella.com | jq
{
  "domain": "umbrella.com",
  "rank": 83069
}

Looking up the domain at a given rank

$ curl -s http://u1m.jn.gl/rank/22923 | jq
{
  "domain": "news.ycombinator.com",
  "rank": 21675
}

The source code for this service is available on Github
